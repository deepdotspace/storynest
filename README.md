# storynest

A web app that turns a simple idea ("a curious fox who learns to share, for a
4-year-old") into a finished, illustrated children's storybook with AI-generated
art, narration audio, and a dedicated reader mode. Built with the
[DeepSpace SDK](https://github.com/deepdotspace).

**Live: [storynest.app.space](https://storynest.app.space)**

## Features

- AI-generated illustrated storybooks from a single prompt
- Per-page art, narration text, and read-aloud audio
- Create wizard: characters, lesson, age band, page count, and art style
- Full-screen reader with auto-playing narration and page flipping
- Per-page re-rolls plus inline text editing
- Public, shareable book reads
- Credits and subscriptions

## Development

